# Generative AI: Customer Transactions & Survival Analysis Evaluation 🚀

## 📌 Project Overview
This repository contains an end-to-end analytics and automation pipeline that pairs **Python-based data diagnostics** with **Generative AI Prompt Engineering**. The framework ingests raw customer transaction data and multi-variable survival logs, cleans structural inconsistencies, runs exploratory statistical modeling, and leverages Large Language Models (LLMs) to automatically synthesize complex metrics into polished, stakeholder-ready executive briefs.

The core goal is to eliminate manual analysis overhead by automating the translation of deep data pipelines into actionable business intelligence.

---

## 🛠️ Tech Stack & Architecture
*   **Data Diagnostics & Modeling:** Python, Pandas, NumPy (Exploratory Data Analysis, data scrubbing, statistical mapping).
*   **AI Automation & Orchestration:** Advanced Prompt Engineering, LLM Evaluation Frameworks, System Prompt Optimization.
*   **Outputs & Reporting:** Automated Markdown/Text synthesis for cross-functional management documentation.

---

## ⚙️ Core Pipeline Architecture

### 1. Python Data Diagnostics Layer
*   **Exploratory Data Analysis (EDA):** Leveraged Pandas and NumPy to ingest messy customer transaction datasets, isolate outliers, and compute core descriptive statistics.
*   **Structural Cleaning:** Built reusable Python functions to catch missing values, fix mismatched data types, and normalize timestamp intervals for time-to-event/survival analysis parameters.

### 2. Generative AI Prompt Optimization Layer
*   **Context-Rich Prompt Engineering:** Designed structured, few-shot prompt templates that force the LLM to act as an elite Business Intelligence Director.
*   **Guardrailed Summarization:** Programmed structural constraints to prevent AI hallucination, ensuring that output summaries strictly rely on the verified statistical coefficients generated in the Python layer.
*   **Automated Workflow Synthesis:** Created repeatable markdown structures so the pipeline dynamically formats statistical insights into clean executive briefing notes.
